# HL_Fabric_Doc-Jp

Hyperledger Fabricドキュメンテーションの日本語翻訳メモリプロジェクトです。

# Scope of Project

このプロジェクトの目的は、Hyperledger Fabricの英語ドキュメントを日本語へ翻訳する際に使用する専門用語や表現の共通化とサポートです。
